# Amazon Clone

A front-end clone of Amazon's homepage built with HTML and CSS.

## Features

- Sticky navbar with search, cart, and account links
- Auto-sliding hero carousel with real Amazon banner images
- Product category cards (4-column grid layout)
- Best Sellers sections with real product images
- Complete footer with all links and Amazon subsidiary brands

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript (carousel)
- Font Awesome icons

## Preview

Open `index.html` in any browser — no build step needed.
